# Grocery Shopping List

[![License: LGPL-3](https://img.shields.io/badge/licence-LGPL--3-blue.png)](http://www.gnu.org/licenses/lgpl-3.0-standalone.html)

Manage grocery shopping lists from the website portal. Create stores, products, and
shopping lists with full tracking capabilities.

## Features

### Store Management

- Create and manage grocery stores
- Store contact information and details
- Track total lists per store

### Product Management

- Create grocery products with categories
- Mark products as favorites
- Track product usage across lists
- Link to Odoo products (admin only)

### Shopping Lists

- Create shopping lists for different stores
- Add products with quantities
- Track purchased items
- Monitor progress with visual indicators
- State management: Draft, In Progress, Completed

### Portal Access

- Full functionality for portal users
- Create and manage lists from website
- Real-time product search
- Store autocomplete

### Backend Management

- Complete backend interface for administrators
- Kanban, list, and form views
- Chatter integration for comments and activities
- Full access control and security rules

## Installation

1. Add the module to your Odoo addons path
2. Update the module list
3. Install the "Grocery Shopping List" module

## Configuration

1. Go to Grocery Lists > Stores
2. Create your grocery stores
3. Optionally create products in Grocery Lists > Products
4. Portal users can access lists from the website portal menu

## Usage

### For Portal Users

1. Log in to the website portal
2. Navigate to "My Account" > "Grocery Lists"
3. Create a new list or view existing ones
4. Add products to your list
5. Start shopping to mark items as purchased
6. Complete the list when finished

### For Administrators

1. Manage stores, products, and lists from the backend
2. Link grocery products to Odoo products for integration
3. Monitor all lists and user activity
4. Use chatter for comments and activities

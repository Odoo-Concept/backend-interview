===============
Shopping List
===============

A simple module for managing shopping lists in Odoo with product catalog integration.

Installation
============

1. Clone the module into your addons directory:

   .. code-block:: bash

      git clone https://github.com/Odoo-Concept/backend-interview.git

2. Restart your Odoo server

3. Go to Applications → Update Apps List

4. Search for "Shopping List" and click Install

Configuration
==============

No configuration is required. The module works out of the box.

Usage
=====

Basic workflow:

1. Go to **Supermarket → Shopping Lists**
2. Create a new list with a name and assign a responsible user
3. Add items with products from your catalog
4. Track the purchase status (Draft → In Progress → Completed)
5. Mark items as purchased as you shop
